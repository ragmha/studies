## Assigning Keys and Values

### `Dot notation`
```javascript
let obj = {};
obj.param = 'new value';
console.log(obj.param); // => new val
```

### `Square bracket notation`
```javascript
let obj = {};
obj['param'] = 'new value';
console.log(obj['param']); // => new val
```
