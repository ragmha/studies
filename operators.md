# Incrmenet (++)

* `x++`, it returns the value before incrementing

* `++x`, it returns the value after incrementing

# Decrmenet (--)

* `x--`, it returns the value before decrementing

* `--x`, it returns the value after decrementing


# Example

```js

// Postfix
var x = 3;
y = x--; // y = 3, x =2

// Prefix
var a = 2;
b = --a; //a=1, b=1

```
